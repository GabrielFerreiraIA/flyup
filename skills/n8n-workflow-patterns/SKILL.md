# n8n Workflow Patterns

## 5 Core Patterns
1. **Webhook Processing**: Triggered by external events.
2. **HTTP API Integration**: Fetching/sending data to services.
3. **Database Operations**: Syncing or processing records.
4. **AI Assistant**: Using Agents and LLMs.
5. **Scheduled Tasks**: Periodic cleanups or reports.

## Checklist
- [ ] Use Webhook for real-time triggers.
- [ ] Add Error Trigger node.
- [ ] Use Set node for data transformation.
- [ ] Batch operations for large data sets.
- [ ] Validate workflow structure.
