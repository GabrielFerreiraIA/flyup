# Flyup 2.0 Core Skill

This skill contains the fundamental design principles and technical requirements for the Flyup 2.0 Skydive Agency platform.

## Design Identity
- **Primary Color**: Neon Green (#39FF14) - used for highlights, icons, and primary buttons.
- **Background**: Deep Black (#050505) and Dark Surface (#101010).
- **Aesthetic**: Cinematic, high-performance, and safe.
- **Typography**: 
    - Headers: Outfit (Bold/Black, often italicized for movement).
    - Body: Inter (Clean and readable).
- **Core Motto**: "Tem que saber voar."

## Technical Stack
- **Framework**: Next.js 15 (App Router).
- **Styling**: Tailwind CSS 4.
- **Animations**: GSAP (ScrollTrigger, Context for cleanup).
- **Components**: Standardized in `src/components/ui` and `src/components/sections`.

## Writing Principles
- Use active, powerful, and professional Portuguese.
- Focus on safety, adrenaline, and authority.
- Avoid "cheap" marketing; use "premium experience" positioning.
